# Experimental Socket.io game

Just an experimental game.

To install and run:

```bash
# Clone and cd into this directory
npm install
node server.js
# Check out your browser and go to localhost:(port).
```

You'll need to be using Node 6 - `nvm install 6; nvm use 6`.
